HTML Entity 코드는 HTML 문서에서 특수 문자(공백,<, >, & 등)을 표현하기 위한 특별한 문자열. 브라우저는 이 코드를 해당하는 문자로 렌더링함.

HTML 에서는 일부 기호들이 태그나 특별한 의미를 나타내는 데 사용되는데 예를들어 < 와 >기호의 경우 HTML태그의 시작과 끝을 나타내며 " 의 경우 속성값을 둘러싸는 데 사용됨.

이러한 문자를 직접 사용하면 HTML 파싱 오류가 발생할수 있기 때문에 특수문자코드(Entity Code)로 변환해서 사용해야함

    자주 쓰이는 Entity Code
  1. 공백 - &nbsp;
  2. < - &lt;
  3. > - &gt;
  4. & - &amp;
  5. " - &quot;
  6. ' - &apos;
  7. © - &copy;

기본적으로 굵은 글꼴이 적용되며 스크린 리더를 사용하여 컴퓨터 화면을 낭독할 때 <strong>태그로 강조된 텍스트를 강조하여 읽어줌

<b>태그

<b>태그는 <strong>태그와 비슷하지만 <strong>태그는 의미를 강조하는 반면 <b>태그는 단순히 시각적으로 강조하는 용도로 사용됨

따라서 <b>태그는 스크린 리더와 같은 보조 기술에서 특별한 의미를 가지지 않음
